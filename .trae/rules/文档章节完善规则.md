# 文档章节完善规则

当用户说"联网搜索相关内容 完善[某]章节"时，规则如下：

## 主章节完善规则

如果用户要求完善**主章节**（如第一章、第二章），则：
- 只添加主章节序号和章节标题
- 不添加任何子章节
- 不填充任何内容

## 子章节完善规则

如果用户要求完善**子章节**（如1.1节、2.3节），则：
- 只添加子章节序号和章节标题
- 不填充任何内容

## 示例

用户说"完善第一章" → 只输出：
```markdown
## 1. 什么是注意力机制 🤔
```

用户说"完善1.1章节" → 只输出：
```markdown
### 1.1 注意力机制的定义 📝
```

---

**核心原则**：用户说"完善章节"时，默认只需要章节骨架（序号+标题），不需要内容。填充内容需要用户明确要求。
