# 参考资料规范

## 联网搜索后的参考资料

在文档中引用搜索结果时，必须在引用处（内容紧跟其后）添加所有相关的参考资料链接。

### 格式要求

- 如果有高质量的参考资料，全部添加
- 如果没有高质量的，则找出最好的（拔尖的）添加
- **紧跟引用内容后面**（内容写完后下一行就是参考资料），不要放在章节末尾

### 高质量资料判断标准

**1. 来源权威性**
- 知名学术平台、专业出版社的出版物
- 顶尖高校或研究机构学者的文章
- 核心期刊、权威会议论文

**2. 内容准确性**
- 内容详实、表达清晰
- 有据可查的观点
- 研究方法严谨

**3. 实践意义**
- 具有现实应用价值
- 解决了实际问题
- 具有启发性和可操作性

**4. 时效性**
- 对于技术类内容，优先选择最新的
- 注意区分内容是否过时

### 链接格式

单个链接：
```markdown
- [标题-平台名](URL)
```
例如：`- [激活函数原理 -- CSDN](https://blog.csdn.net/...)`

多个链接：
```markdown
内容段落（联网搜索后编写的内容）...

（此处填写具体知识点名称）参考资料：
- [标题1 -- 平台名](URL1)
- [标题2 -- 平台名](URL2)
```

## 文档内部引用

### 文档间引用

> **注意**：每个文档文件夹下都有一个专门记录文档链接的文件（如 `00-Transformer技术专栏链接目录.md`），添加相关文档链接时需要去该文件查询。

- **文档内引用其他文档**：当涉及其他文档的知识点时，必须附上链接
  - 示例：`在[08a. 检索算法与策略-稠密检索技术](https://juejin.cn/...)中我们学习了...`
  - 跨文档引用必须具体到章节，方便读者定位
  - 链接格式：掘金链接 / CSDN链接

## 引用规则

- 不可以直接复制他人的语句，要用自己的话重新表述内容
- 如果需要引用内容，必须按照 MLA 格式来标注
