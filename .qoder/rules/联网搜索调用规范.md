# 联网搜索调用规范 🔍

## 时间上下文

**当前时间**：2026年6月

在编写搜索query时，应注意当前年份，确保搜索最新版本的技术信息。

---

## 核心原则

搜索时应传入**具体的技术术语和关键词组合**，而非口语化表达或疑问句。

---

## Query 编写规范

### ✅ 正确示例

**搜索技术实践**：
```python
query="Python async await 协程 最佳实践 2024"
query="Transformer 多头注意力机制 优化"
query="Spring Boot 3.0 迁移指南 JDK17"
```

**搜索最新版本**：
```python
query="PyTorch 2.0 scaled_dot_product_attention FlashAttention"
query="React 19 新特性 Server Components"
query="Kubernetes 1.30 版本更新"
```

**搜索问题解决方案**：
```python
query="Docker CVE-2024 安全漏洞 修复方案"
query="PyTorch CUDA out of memory 显存优化 技巧"
query="MySQL 慢查询 索引优化 性能调优"
```

**搜索经典理论**：
```python
query="TCP 三次握手 四次挥手 原理"
query="HTTP HTTPS 协议 工作原理"
query="Transformer self-attention 注意力机制 原理"
```

### ❌ 错误示例

```python
# 太口语化
query="python怎么异步"
query="springboot新版本怎么用"

# 缺少关键细节
query="pytorch attention"
query="docker漏洞"

# 疑问句形式
query="什么是TCP三次握手"
query="如何优化MySQL性能"
```

---

## Query 编写原则

1. **技术术语优先**：使用官方文档中的标准术语
   - ✅ `asyncio 协程 并发编程`
   - ❌ `python异步怎么写`

2. **版本信息明确**：涉及版本时明确标注
   - ✅ `Python 3.12 新特性 2024`
   - ❌ `python最新版本`

3. **关键词组合**：3-5个核心关键词，用空格分隔
   - ✅ `Transformer 多头注意力 优化 最佳实践`
   - ❌ `transformer`

4. **避免疑问句**：使用陈述式关键词而非问题形式
   - ✅ `TCP 三次握手 原理`
   - ❌ `什么是TCP三次握手`

## 时间范围选择

根据搜索内容类型选择 `timeRange`：

| 场景 | 推荐值 | 示例 |
|------|--------|------|
| 最新bug/安全漏洞 | `OneDay` 或 `OneWeek` | `"CVE-2024 漏洞修复"` |
| 快速迭代的技术 | `OneMonth` | `"React 19 新特性"` |
| 年度总结/版本更新 | `OneYear` | `"Python 3.12 特性 2024"` |
| 经典知识/基础概念 | `NoLimit` | `"TCP三次握手原理"` |
| 技术演进/发展历史 | `NoLimit` | `"Transformer发展历史"` |

## 搜索策略

### 标准流程

```
1. search_web 获取多个来源摘要
   ↓
2. 筛选高质量结果（官方文档、权威博客、最新内容）
   ↓
3. fetch_content 深入获取重要链接的详细内容
   ↓
4. 综合多个来源，用自己的话重新表述
   ↓
5. 按参考链接规范添加参考资料
```

### 高质量结果判断

**核心标准**（按优先级）：
1. **内容准确性** - 详实清晰、代码可运行、无技术错误
2. **实践价值** - 有可操作示例、真实场景案例
3. **深度与完整性** - 系统讲解、涵盖关键细节和陷阱
4. **时效性** - 技术内容优先最新的
5. **来源权威性** - 官方文档、知名平台（辅助参考）
6. **表达易懂性** - 通俗易懂、有类比图解（加分项）

---

## 注意事项

1. **搜索词要具体**：避免宽泛词汇
2. **时间范围要合理**：不盲目使用最新时间
3. **多渠道验证**：重要信息交叉验证
4. **内容复述**：不直接复制，用自己的话表述
5. **链接格式统一**：严格按参考链接规范
6. **质量优先**：宁缺毋滥
