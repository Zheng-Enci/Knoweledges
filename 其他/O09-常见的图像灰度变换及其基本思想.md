## 1. 灰度变换基础概念 📖

> 📘 本章介绍灰度变换的定义、数学表达和基本思想

### 1.1 什么是灰度变换？🤔

**灰度变换（Grayscale Transformation）** 是图像增强的一种重要手段，属于空间域处理方法。它的核心思想是：**直接对图像的每个像素进行灰度值映射，以改善图像的视觉效果或突出特定特征**。

**数学表达** 📐：

灰度变换可以表示为一个函数：

$$
s = T(r)
$$

其中：
- $r$ 是输入图像在位置 $(x, y)$ 的灰度值
- $s$ 是输出图像在相同位置的灰度值
- $T$ 是灰度变换函数，定义了从输入到输出的映射规则

**直观类比** 🎨：想象灰度变换是一个"调色板"
- 输入图像的每个像素值就像原材料
- 变换函数 $T$ 就是加工规则
- 输出图像就是按照规则加工后的成品

### 1.2 灰度变换的分类 📋

根据变换函数 $T$ 的性质，灰度变换可以分为三大类：

| 类型 | 特点 | 典型方法 |
|------|------|---------|
| **线性变换** 📏 | 变换函数是线性的，保持灰度级的比例关系 | 反转变换、对比度调整、灰度平移 |
| **非线性变换** 📐 | 变换函数是非线性的，可以有选择地增强特定灰度范围 | 对数变换、伽马变换、幂律变换 |
| **直方图变换** 📊 | 基于图像灰度分布统计特性进行自适应变换 | 直方图均衡化、直方图规定化 |

## 2. 线性灰度变换 📏

> 📏 本章学习线性灰度变换的核心方法

### 2.1 反转变换（Image Negative）🔄

**反转变换** 是最简单的线性变换，它将图像的亮区域变暗，暗区域变亮。

**变换公式** 📐：

$$
s = L - 1 - r
$$

其中：
- $r$ 是输入灰度值
- $s$ 是输出灰度值
- $L$ 是灰度级总数（对于 8 位图像，$L = 256$）

**直观类比** 🎨：反转变换就像照片的"底片"效果
- 原来的白色（255）变成黑色（0）
- 原来的黑色（0）变成白色（255）
- 中间的灰度值也相应反转

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢

"""反转变换实现

参数:
    img: 输入灰度图像，形状 [H, W]
    
返回:
    output: 反转后的图像，形状 [H, W]
    
示例:
    reversed_img = reverse(gray_img)
"""
def reverse(img):
    output = 255 - img                                      # 应用反转公式 s = 255 - r，数据流动：img[H,W] → output[H,W]
    return output

# 读取图像并转换为灰度图
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取图像，数据流动：文件 → img[H,W] 🖼️
reversed_img = reverse(img)                                 # 应用反转变换，数据流动：img[H,W] → reversed_img[H,W] 🔄

# 显示结果
cv2.imshow('Original', img)                                 # 显示原图 🖼️
cv2.imshow('Reversed', reversed_img)                        # 显示反转图 🔄
cv2.waitKey(0)                                              # 等待按键 ⏸️
cv2.destroyAllWindows()                                     # 关闭所有窗口 🚪
```

**应用场景** 🎯：
- 增强嵌入在暗区域中的白色或灰色细节
- 医学图像（如 X 光片、乳房 X 光片）的分析
- 艺术效果和视觉反转

### 2.2 对比度调整 📊

**对比度调整** 通过线性缩放灰度值来增强或减弱图像的对比度。

**变换公式** 📐：

$$
s = a \times r + b
$$

其中：
- $r$ 是输入灰度值
- $s$ 是输出灰度值
- $a$ 是增益系数（控制对比度）
- $b$ 是偏移量（控制亮度）

**参数含义** 📋：

| 参数 | 效果 | 示例 |
|------|------|------|
| $a > 1$ | 增强对比度，拉伸灰度范围 | $a = 1.5$，对比度增强 50% |
| $0 < a < 1$ | 减弱对比度，压缩灰度范围 | $a = 0.8$，对比度减弱 20% |
| $a = 1$ | 保持对比度不变 | 恒等变换 |
| $b > 0$ | 增加亮度，灰度整体上移 | $b = 50$，整体变亮 |
| $b < 0$ | 降低亮度，灰度整体下移 | $b = -30$，整体变暗 |

**直观类比** 🎨：对比度调整就像调节音响的音量
- $a$ 相当于音量旋钮：放大（$a > 1$）或缩小（$a < 1$）声音的差异
- $b$ 相当于基础音量：整体调高或调低

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢

"""对比度调整实现

参数:
    img: 输入灰度图像，形状 [H, W]
    a: 增益系数（控制对比度），默认 1.5
    b: 偏移量（控制亮度），默认 0
    
返回:
    output: 调整后的图像，形状 [H, W]
    
示例:
    enhanced = contrast_adjust(gray_img, a=1.5, b=0)
"""
def contrast_adjust(img, a=1.5, b=0):
    output = a * img + b                                    # 应用线性变换公式 s = a*r + b，数据流动：img[H,W] → output[H,W]
    output = np.clip(output, 0, 255)                        # 截断到 [0, 255] 范围，防止溢出，数据流动：output[H,W] → output[H,W]（ clipped）
    output = np.uint8(output)                               # 转换为 8 位无符号整数，数据流动：output[H,W] float → output[H,W] uint8 🔢
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# 增强对比度（a=1.5）
enhanced = contrast_adjust(img, a=1.5, b=0)                 # 对比度增强 50%，数据流动：img[H,W] → enhanced[H,W] 📊

# 减弱对比度（a=0.8）
weakened = contrast_adjust(img, a=0.8, b=0)                 # 对比度减弱 20%，数据流动：img[H,W] → weakened[H,W] 📉

# 增加亮度（b=50）
brighter = contrast_adjust(img, a=1.0, b=50)                # 整体变亮 50，数据流动：img[H,W] → brighter[H,W] ☀️
```

## 3. 非线性灰度变换 📐

> 📐 本章掌握对数变换和伽马变换的原理

### 3.1 对数变换（Log Transformation）📈

**对数变换** 可以拉伸范围较窄的低灰度值，同时压缩范围较宽的高灰度值。

**变换公式** 📐：

$$
s = c \times \log(1 + r)
$$

其中：
- $r$ 是输入灰度值
- $s$ 是输出灰度值
- $c$ 是缩放常数，用于调整输出范围
- $\log$ 是自然对数（以 $e$ 为底）或常用对数（以 10 为底）

**为什么要加 1？** 🤔

$r + 1$ 的作用是防止 $r = 0$ 时 $\log(0)$ 无定义。加 1 后，函数向左平移一个单位，确保所有输出值都大于 0。

**直观类比** 🎨：对数变换就像"放大镜"
- 对暗区域（低灰度值）进行"放大"，拉伸细节
- 对亮区域（高灰度值）进行"压缩"，避免过曝

**典型应用** 🌟：

对数变换的经典应用是**傅里叶频谱的显示**。傅里叶变换后的频谱值范围可能非常宽（如 $0$ 到 $10^7$），直接显示会导致大部分细节不可见。对数变换可以将这个宽范围压缩到可视范围（如 $0$ 到 $17.5$）。

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢
import matplotlib.pyplot as plt                             # 导入 Matplotlib 库 📊

"""对数变换实现

参数:
    img: 输入灰度图像，形状 [H, W]
    c: 缩放常数，默认 42
    
返回:
    output: 对数变换后的图像，形状 [H, W]
    
示例:
    log_img = log_transform(gray_img, c=42)
"""
def log_transform(img, c=42):
    output = c * np.log(1.0 + img.astype(np.float32))       # 应用对数变换公式 s = c*log(1+r)，数据流动：img[H,W] → output[H,W] float 📈
    output = np.uint8(np.clip(output + 0.5, 0, 255))        # 四舍五入并截断到 [0, 255]，转换为 uint8，数据流动：output[H,W] → output[H,W] uint8 🔢
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# 应用对数变换
log_img = log_transform(img, c=42)                          # 对数变换，拉伸暗部细节，数据流动：img[H,W] → log_img[H,W] 📈

# 可视化变换函数
x = np.arange(0, 256, 0.01)                                 # 生成 x 轴数据 [0, 255]，步长 0.01 🔢
y = 42 * np.log(1 + x)                                      # 计算对数变换曲线 y = 42*log(1+x)，数据流动：x[25600] → y[25600] 📈

plt.figure(figsize=(8, 6))                                  # 创建画布，尺寸 8×6 英寸 🖼️
plt.plot(x, y, 'r', linewidth=1)                            # 绘制对数变换曲线，红色线条 📈
plt.title('Log Transformation Function')                    # 设置标题 🏷️
plt.xlim([0, 255])                                          # 设置 x 轴范围 [0, 255] ↔️
plt.ylim([0, 255])                                          # 设置 y 轴范围 [0, 255] ↕️
plt.xlabel('Input Gray Level (r)')                          # 设置 x 轴标签 ➡️
plt.ylabel('Output Gray Level (s)')                         # 设置 y 轴标签 ⬆️
plt.grid(True, linestyle=':', alpha=0.7)                    # 添加网格，虚线样式，透明度 0.7 📐
plt.show()                                                  # 显示图像 🖼️
```

**应用场景** 🎯：
- 扩展图像中的暗像素值，增强暗部细节
- 傅里叶频谱的可视化
- 处理动态范围很宽的图像（如 HDR 图像压缩）

### 3.2 伽马变换（Gamma / Power-Law Transformation）⚡

**伽马变换**（又称幂律变换）通过幂函数对灰度值进行非线性映射。

**变换公式** 📐：

$$
s = c \times r^{\gamma}
$$

其中：
- $r$ 是输入灰度值（通常归一化到 $[0, 1]$）
- $s$ 是输出灰度值
- $c$ 是缩放常数
- $\gamma$ 是伽马值（幂指数）

**伽马值的影响** 📋：

| $\gamma$ 值 | 效果 | 适用场景 |
|-----------|------|---------|
| $\gamma < 1$ | 拉伸低灰度区域，压缩高灰度区域 | 图像整体偏暗，需要提亮暗部 |
| $\gamma = 1$ | 恒等变换（线性） | 不改变图像 |
| $\gamma > 1$ | 拉伸高灰度区域，压缩低灰度区域 | 图像过亮（"冲淡"外观），需要压暗 |

**直观类比** 🎨：伽马变换就像"智能调光器"
- $\gamma < 1$：像夜间模式，提亮暗部但不过曝亮部
- $\gamma > 1$：像阳光模式，增强亮部对比度但保留暗部

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢
import matplotlib.pyplot as plt                             # 导入 Matplotlib 库 📊

"""伽马变换实现

参数:
    img: 输入灰度图像，形状 [H, W]
    c: 缩放常数，默认 1.0
    gamma: 伽马值（幂指数），默认 0.5（提亮暗部）
    
返回:
    output: 伽马变换后的图像，形状 [H, W]
    
示例:
    gamma_img = gamma_transform(gray_img, c=1.0, gamma=0.5)
"""
def gamma_transform(img, c=1.0, gamma=0.5):
    # 构建查找表（LUT），数据流动：循环 256 次 → lut[256] 🔢
    lut = np.zeros(256, dtype=np.float32)                   # 初始化查找表，大小 256，浮点类型 🔢
    for i in range(256):                                    # 遍历 0 到 255 的灰度值 🔄
        lut[i] = c * (i / 255.0) ** gamma                   # 计算伽马变换值 s = c*(r/255)^γ，数据流动：i → lut[i] ⚡
    
    # 应用查找表，数据流动：img[H,W] + lut[256] → output[H,W] 🖼️
    output = cv2.LUT(img, lut)                              # 使用 OpenCV LUT 函数高效应用变换 ⚡
    output = np.uint8(np.clip(output * 255.0 + 0.5, 0, 255)) # 反归一化到 [0, 255]，四舍五入并转换类型 🔢
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# γ < 1：提亮暗部（适合偏暗图像）
gamma_dark = gamma_transform(img, c=1.0, gamma=0.4)         # γ=0.4，拉伸暗部细节，数据流动：img[H,W] → gamma_dark[H,W] 🌙

# γ > 1：压暗暗部（适合过亮图像）
gamma_bright = gamma_transform(img, c=1.0, gamma=2.2)       # γ=2.2，增强亮部对比度，数据流动：img[H,W] → gamma_bright[H,W] ☀️

# 可视化不同伽马值的变换曲线
plt.figure(figsize=(10, 6))                                 # 创建画布，尺寸 10×6 英寸 🖼️
x = np.linspace(0, 1, 256)                                  # 生成归一化 x 轴 [0, 1]，256 个点 🔢

gammas = [0.2, 0.4, 0.6, 1.0, 1.5, 2.2, 3.0]               # 不同的伽马值列表 ⚡
for g in gammas:                                            # 遍历每个伽马值 🔄
    y = x ** g                                              # 计算变换曲线 y = x^γ，数据流动：x[256] → y[256] 📈
    plt.plot(x, y, label=f'γ={g}')                          # 绘制曲线，添加图例标签 📊

plt.title('Gamma Transformation Curves')                    # 设置标题 🏷️
plt.xlabel('Normalized Input (r)')                          # 设置 x 轴标签 ➡️
plt.ylabel('Normalized Output (s)')                         # 设置 y 轴标签 ⬆️
plt.legend()                                                # 显示图例 📋
plt.grid(True, linestyle=':', alpha=0.7)                    # 添加网格 📐
plt.show()                                                  # 显示图像 🖼️
```

**应用场景** 🎯：
- **显示器校正**：CRT 显示器和 LCD 屏幕的亮度响应不是线性的，需要伽马校正
- **医学图像增强**：调整 X 光片、CT 图像的对比度
- **摄影后期处理**：修正曝光不足或过度的照片

## 4. 分段线性变换 📊

> 📊 本章学习对比度拉伸和灰度级分层

### 4.1 对比度拉伸（Contrast Stretching）🎯

**对比度拉伸** 通过分段线性函数有选择地增强感兴趣的灰度区域，同时抑制不感兴趣的区域。

**变换公式** 📐：

对比度拉伸使用分段线性函数，通常由 3 段组成：

$$
s = \begin{cases}
\frac{y_1}{x_1} \times r, & 0 \leq r < x_1 \\
\frac{y_2 - y_1}{x_2 - x_1} \times (r - x_1) + y_1, & x_1 \leq r < x_2 \\
\frac{255 - y_2}{255 - x_2} \times (r - x_2) + y_2, & x_2 \leq r \leq 255
\end{cases}
$$

其中：
- $(x_1, y_1)$ 和 $(x_2, y_2)$ 是两个控制点
- $x_1, x_2$ 是输入灰度的分界点
- $y_1, y_2$ 是对应的输出灰度值

**直观类比** 🎨：对比度拉伸就像"局部放大镜"
- 你感兴趣的灰度范围（如 $[x_1, x_2]$）会被"拉伸"到更大的输出范围 $[y_1, y_2]$
- 不感兴趣的区域会被"压缩"

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢
import matplotlib.pyplot as plt                             # 导入 Matplotlib 库 📊

"""分段线性变换（对比度拉伸）实现

参数:
    img: 输入灰度图像，形状 [H, W]
    x1: 第一个控制点的输入灰度值，默认 100
    x2: 第二个控制点的输入灰度值，默认 160
    y1: 第一个控制点的输出灰度值，默认 30
    y2: 第二个控制点的输出灰度值，默认 230
    
返回:
    output: 对比度拉伸后的图像，形状 [H, W]
    
示例:
    stretched = contrast_stretch(gray_img, x1=100, x2=160, y1=30, y2=230)
"""
def contrast_stretch(img, x1=100, x2=160, y1=30, y2=230):
    # 构建查找表（LUT），数据流动：循环 256 次 → lut[256] 🔢
    lut = np.zeros(256, dtype=np.float32)                   # 初始化查找表，大小 256 📊
    for i in range(256):                                    # 遍历 0 到 255 的灰度值 🔄
        if i < x1:                                          # 第一段：[0, x1)
            lut[i] = (y1 / x1) * i                          # 线性映射 s = (y1/x1)*r，数据流动：i → lut[i] 📏
        elif i < x2:                                        # 第二段：[x1, x2)
            lut[i] = ((y2 - y1) / (x2 - x1)) * (i - x1) + y1 # 线性映射 s = slope*(r-x1)+y1，数据流动：i → lut[i] 📈
        else:                                               # 第三段：[x2, 255]
            lut[i] = ((255 - y2) / (255 - x2)) * (i - x2) + y2 # 线性映射 s = slope*(r-x2)+y2，数据流动：i → lut[i] 📉
    
    # 应用查找表，数据流动：img[H,W] + lut[256] → output[H,W] 🖼️
    output = cv2.LUT(img, lut)                              # 使用 OpenCV LUT 高效应用变换 ⚡
    output = np.uint8(np.clip(output + 0.5, 0, 255))        # 四舍五入并截断到 [0, 255]，转换为 uint8 🔢
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# 应用对比度拉伸
stretched = contrast_stretch(img, x1=100, x2=160, y1=30, y2=230) # 拉伸 [100,160] 到 [30,230] 🎯

# 可视化变换函数
plt.figure(figsize=(8, 6))                                  # 创建画布，尺寸 8×6 英寸 🖼️
plt.plot([0, 100, 160, 255], [0, 30, 230, 255], 'b-', linewidth=2) # 绘制分段线性变换曲线 🔵
plt.plot([100, 100, 0], [0, 30, 30], 'r--', alpha=0.7)     # 绘制辅助线（虚线），标注控制点 1 🔴
plt.plot([160, 160, 0], [0, 230, 230], 'r--', alpha=0.7)   # 绘制辅助线（虚线），标注控制点 2 🔴
plt.title('Contrast Stretching Function')                   # 设置标题 🏷️
plt.xlim([0, 255])                                          # 设置 x 轴范围 [0, 255] ↔️
plt.ylim([0, 255])                                          # 设置 y 轴范围 [0, 255] ↕️
plt.xlabel('Input Gray Level (r)')                          # 设置 x 轴标签 ➡️
plt.ylabel('Output Gray Level (s)')                         # 设置 y 轴标签 ⬆️
plt.grid(True, linestyle=':', alpha=0.7)                    # 添加网格 📐
plt.show()                                                  # 显示图像 🖼️
```

**应用场景** 🎯：
- 增强特定灰度范围的细节（如增强中等亮度区域）
- 抑制噪声或不感兴趣的背景
- 工业检测中的缺陷增强

### 4.2 灰度级分层（Gray-Level Slicing）🔪

**灰度级分层** 将特定灰度范围的像素突出显示，其他像素保持或压暗。

**变换公式** 📐：

$$
s = \begin{cases}
\text{high}, & r_1 \leq r \leq r_2 \\
\text{low}, & \text{其他}
\end{cases}
$$

或保留背景：

$$
s = \begin{cases}
\text{high}, & r_1 \leq r \leq r_2 \\
r, & \text{其他}
\end{cases}
$$

其中：
- $[r_1, r_2]$ 是感兴趣的灰度范围
- `high` 是突出显示的灰度值（如 255，白色）
- `low` 是其他区域的灰度值（如 0，黑色）

**Python 实现** 💻：

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢

"""灰度级分层实现

参数:
    img: 输入灰度图像，形状 [H, W]
    layer1: 第一个分层阈值，默认 30
    layer2: 第二个分层阈值，默认 60
    value1: 范围外的灰度值，默认 10
    value2: 范围内的灰度值，默认 250
    
返回:
    output: 灰度级分层后的图像，形状 [H, W]
    
示例:
    sliced = gray_level_slicing(gray_img, layer1=30, layer2=60)
"""
def gray_level_slicing(img, layer1=30, layer2=60, value1=10, value2=250):
    lut = np.zeros(256, dtype=np.uint8)                     # 初始化查找表，大小 256，uint8 类型 🔢
    for i in range(256):                                    # 遍历 0 到 255 的灰度值 🔄
        if i >= layer2:                                     # 灰度值 >= layer2
            lut[i] = value1                                 # 设置为 value1（低灰度），数据流动：i → lut[i] = 10 🔽
        elif i >= layer1:                                   # layer1 <= 灰度值 < layer2
            lut[i] = value2                                 # 设置为 value2（高灰度），数据流动：i → lut[i] = 250 🔼
        else:                                               # 灰度值 < layer1
            lut[i] = value1                                 # 设置为 value1（低灰度），数据流动：i → lut[i] = 10 🔽
    
    output = cv2.LUT(img, lut)                              # 应用查找表，数据流动：img[H,W] + lut[256] → output[H,W] 🖼️
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# 应用灰度级分层（突出 [30, 60] 范围）
sliced = gray_level_slicing(img, layer1=30, layer2=60, value1=10, value2=250) # 突出中等灰度区域 🔪
```

**应用场景** 🎯：
- 卫星图像中突出特定地物类型
- 医学图像中突出特定组织
- 工业检测中突出缺陷区域

## 5. 直方图均衡化 📈

> 📈 本章深入理解基于统计的自动增强方法

### 5.1 什么是直方图均衡化？🤔

**直方图均衡化（Histogram Equalization）** 是一种增强图像对比度的方法，其核心思想是：**将图像的灰度直方图分布通过累积分布函数（CDF）变换为近似均匀分布，从而增强图像的整体对比度**。

**直观类比** 🎨：直方图均衡化就像"自动调色师"
- 它会自动分析图像的灰度分布
- 将集中在某个区间的灰度值"均匀散布"到整个灰度范围
- 不需要手动设置参数，自适应性强

### 5.2 数学原理 📐

直方图均衡化的理论基础是**累积分布函数（Cumulative Distribution Function, CDF）**。

**概率密度函数（PDF）** 📊：

对于灰度级 $r_k$（$k = 0, 1, \ldots, L-1$），其概率为：

$$
p_r(r_k) = \frac{n_k}{N}
$$

其中：
- $n_k$ 是灰度级为 $r_k$ 的像素数量
- $N$ 是图像的总像素数量
- $L$ 是灰度级总数（通常 $L = 256$）

**累积分布函数（CDF）** 📈：

$$
CDF(r_k) = \sum_{i=0}^{k} p_r(r_i)
$$

**均衡化映射公式** 📐：

$$
s_k = (L - 1) \times \sum_{i=0}^{k} p_r(r_i) = (L - 1) \times CDF(r_k)
$$

其中：
- $s_k$ 是映射后的输出灰度值
- $L - 1 = 255$（对于 8 位图像）

**为什么使用 CDF？** 🤔

CDF 满足两个关键条件：✅
1. **单调递增**：保证映射后灰度级的大小关系不变（亮的仍然亮，暗的仍然暗）
2. **值域在 $[0, 1]$**：保证映射后的灰度值在合法范围内

### 5.3 公式推导 📝

设 $r$ 为归一化后的输入图像灰度（$r \in [0, 1]$），$s$ 为输出图像灰度（$s \in [0, 1]$）。

若随机变量 $r$ 的概率密度函数为 $p_r(r)$，且随机变量 $s$ 为 $r$ 的函数，则有：

$$
\int_0^s p_s(s) ds = \int_0^r p_r(r) dr
$$

两边对 $s$ 求导数：

$$
p_s(s) = p_r(r) \frac{dr}{ds}
$$

当输出图像的灰度均匀分布时，$p_s(s) = 1$（常数），上式变为：

$$
p_r(r) = \frac{ds}{dr}
$$

等式两边对 $r$ 积分：

$$
s = \int_0^r p_r(r) dr
$$

这就是 CDF 的定义！因此，**均衡化映射就是输入灰度的累积分布函数**。

对于离散图像，公式变为：

$$
s_k = \sum_{i=0}^{k} p_r(r_i)
$$

最终输出需要乘以 $255$（反归一化）：

$$
s_k = 255 \times \sum_{i=0}^{k} p_r(r_i)
$$

### 5.4 Python 实现 💻

```python
import cv2                                                  # 导入 OpenCV 库 🖼️
import numpy as np                                          # 导入 NumPy 库 🔢
import matplotlib.pyplot as plt                             # 导入 Matplotlib 库 📊

"""计算图像的概率密度函数（PDF）

参数:
    img: 输入灰度图像，形状 [H, W]
    
返回:
    pdf: 概率密度数组，长度 256，pdf[k] 表示灰度 k 的概率
    
示例:
    pdf = get_pdf(gray_img)
"""
def get_pdf(img):
    total = img.shape[0] * img.shape[1]                     # 计算图像总像素数，数据流动：img[H,W] → total（标量）🔢
    pdf = np.array([np.sum(img == i) / total for i in range(256)]) # 计算每个灰度级的概率，数据流动：img[H,W] → pdf[256] 📊
    return pdf

"""直方图均衡化实现（核心代码）

参数:
    img: 输入灰度图像，形状 [H, W]
    
返回:
    output: 均衡化后的图像，形状 [H, W]
    
示例:
    equalized = histogram_equalization(gray_img)
"""
def histogram_equalization(img):
    # 1. 求输入图像的概率密度
    Pr = get_pdf(img)                                       # 计算 PDF，数据流动：img[H,W] → Pr[256] 📊
    
    # 2. 构造输出图像（初始化为输入的副本）
    output = np.copy(img)                                   # 复制输入图像，数据流动：img[H,W] → output[H,W] 📋
    
    # 3. 执行直方图均衡化（累积分布函数变换）
    CDF = 0.0                                               # 累积分布值初始化，浮点数 🔢
    for i in range(256):                                    # 遍历 0 到 255 的灰度级 🔄
        CDF = CDF + Pr[i]                                   # 累加概率，计算 CDF(i)，数据流动：CDF += Pr[i] 📈
        output[img == i] = CDF * 255.0                      # 应用映射 s = CDF*255，数据流动：output[H,W]（部分更新）🎯
    
    output = np.uint8(np.clip(output, 0, 255))              # 截断到 [0, 255] 并转换为 uint8，数据流动：output[H,W] → output[H,W] uint8 🔢
    return output

# 读取图像
img = cv2.imread('input.jpg', cv2.IMREAD_GRAYSCALE)         # 读取灰度图像 🖼️

# 手动实现直方图均衡化
equalized_manual = histogram_equalization(img)              # 手动均衡化，数据流动：img[H,W] → equalized_manual[H,W] 📈

# 使用 OpenCV 内置函数（对比验证）
equalized_cv2 = cv2.equalizeHist(img)                       # OpenCV 均衡化，数据流动：img[H,W] → equalized_cv2[H,W] ⚡

# 可视化结果
fig, axes = plt.subplots(2, 2, figsize=(12, 10))            # 创建 2×2 子图布局，尺寸 12×10 英寸 🖼️

# 原图
axes[0, 0].imshow(img, cmap='gray', vmin=0, vmax=255)       # 显示原图，灰度颜色映射 🖼️
axes[0, 0].set_title('Original Image')                      # 设置标题 🏷️
axes[0, 0].axis('off')                                      # 关闭坐标轴 🚫

# 原图直方图
axes[0, 1].hist(img.flatten(), bins=256, range=[0, 256], color='blue', alpha=0.7) # 绘制原图直方图，256 个 bin 📊
axes[0, 1].set_title('Original Histogram')                  # 设置标题 🏷️
axes[0, 1].set_xlim([0, 256])                               # 设置 x 轴范围 ↔️
axes[0, 1].grid(True, linestyle=':', alpha=0.5)             # 添加网格 📐

# 均衡化后的图像
axes[1, 0].imshow(equalized_manual, cmap='gray', vmin=0, vmax=255) # 显示均衡化后的图像 🖼️
axes[1, 0].set_title('Equalized Image (Manual)')            # 设置标题 🏷️
axes[1, 0].axis('off')                                      # 关闭坐标轴 🚫

# 均衡化后的直方图
axes[1, 1].hist(equalized_manual.flatten(), bins=256, range=[0, 256], color='red', alpha=0.7) # 绘制均衡化后的直方图 📊
axes[1, 1].set_title('Equalized Histogram')                 # 设置标题 🏷️
axes[1, 1].set_xlim([0, 256])                               # 设置 x 轴范围 ↔️
axes[1, 1].grid(True, linestyle=':', alpha=0.5)             # 添加网格 📐

plt.tight_layout()                                          # 自动调整布局 📐
plt.savefig('histogram_equalization.png', dpi=150)          # 保存图片，150 DPI 💾
plt.show()                                                  # 显示图像 🖼️
```

### 5.5 直方图均衡化的特点 📋

**优点** ✅：
- **全自动**：不需要手动设置参数
- **自适应**：根据图像内容自动调整
- **对比度增强显著**：特别适合直方图集中在某个区间的图像

**局限性** ⚠️：
- **可能过度增强**：对某些图像产生不自然的效果
- **噪声放大**：如果图像有噪声，均衡化可能放大噪声
- **全局处理**：不考虑局部特征，可能不适合光照不均匀的图像

**应用场景** 🎯：
- 医学图像增强（X 光片、CT、MRI）
- 卫星图像和航拍图像
- 低对比度图像的自动增强
- 图像预处理（为后续处理做准备）

## 6. 总结 📝

本节我们系统学习了图像灰度变换的核心方法，回顾要点：🎯

| 方法 | 类型 | 公式 | 主要用途 |
|------|------|------|---------|
| **反转变换** 🔄 | 线性 | $s = 255 - r$ | 增强暗区域细节 |
| **对比度调整** 📊 | 线性 | $s = a \times r + b$ | 全局对比度增强/减弱 |
| **对数变换** 📈 | 非线性 | $s = c \times \log(1 + r)$ | 扩展暗部，压缩亮部 |
| **伽马变换** ⚡ | 非线性 | $s = c \times r^{\gamma}$ | 自适应灰度调整 |
| **对比度拉伸** 🎯 | 分段线性 | 分段函数 | 增强特定灰度范围 |
| **直方图均衡化** 📈 | 统计 | $s_k = 255 \times CDF(r_k)$ | 自动对比度增强 |

🔴 **关键理解**：

- 💡 **线性变换** 保持灰度级的比例关系，适合全局调整 📏
- 💡 **非线性变换** 可以有选择地增强特定灰度范围，灵活性更高 📐
- 💡 **直方图均衡化** 是全自动的自适应方法，不需要手动设置参数 📈
- 💡 **查找表（LUT）** 是高效实现灰度变换的关键技术，OpenCV 的 `cv2.LUT` 性能优异 ⚡

**实践建议** 🌟：
- 对于简单调整，使用线性变换（对比度调整）
- 对于复杂场景（如医学图像、低光照图像），优先尝试伽马变换或直方图均衡化
- 实际项目中，可以组合多种方法达到最佳效果
