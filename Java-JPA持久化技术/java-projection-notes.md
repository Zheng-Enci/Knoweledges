# J6A1-JPA投影资料复盘与检查清单

## 资料复盘要点

- Spring Data JPA 投影类型：接口（封闭/开放）、类/record、DTO、动态投影，引用 `参考资料/社区和搜索引擎/信息.md` 中 CSDN、Google 搜索多篇内容。
- 典型场景：只读列表、导出接口、敏感内容裁剪、性能瓶颈表查询、OneToMany 拉取控制、DTO 传输。
- 关键机制：通过接口 getter 映射列、`@Value` SpEL、`select new ...` 构造表达式、Criteria 投影、动态 `<T> T find...`.
- 常见坑：关联集合未加载、开放投影不会裁剪 SQL、OneToMany 需 `LEFT_JOIN`、嵌套接口易触发 N+1。
