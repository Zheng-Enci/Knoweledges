## 1. 前置知识点

在深入学习 Python 字典之前，你需要了解以下基础概念：

- **变量（variable（变量））**：用于存储数据的容器，就像生活中的 **盒子**，可以在里面存放不同的物品
- **数据类型（data type（数据类型））**：Python 中的基本数据类型包括整数、浮点数、字符串等
- **键值对（key-value pair（键值对））**：一种数据结构，由唯一的键（key）和对应的值（value）组成
- **哈希表（hash table（哈希表））**：字典的底层数据结构，提供快速的查找性能

---

💡 **学习建议**：如果你已经掌握了 Python 变量和基本数据类型，就可以开始学习字典了。字典是 Python 最常用的数据结构之一，是后续学习 JSON、API 响应处理等内容的基础。
