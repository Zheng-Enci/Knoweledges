## 1. 矩阵的基本概念 🤔

### 1.1 矩阵的定义 📝

> 本节介绍矩阵的基本定义和核心概念

**矩阵**是一个按照长方阵列排列的数集合。在数学中，矩阵（Matrix）是一个由 m × n 个数排成的 m 行 n 列的矩形表格。

矩阵通常用大写字母表示，如 A、B、C。矩阵中的元素用小写字母表示，如 aᵢⱼ，其中 i 和 j 分别表示元素所在的行和列。

**矩阵的数学形式**：

```
A = | a₁₁ a₁₂ ... a₁ₙ |
    | a₂₁ a₂₂ ... a₂ₙ |
    | ...  ... ... .. |
    | aₘ₁ aₘ₂ ... aₘₙ |
```

这个 m × n 的矩阵可以记为 A = [aᵢⱼ]ₘₙ。

**矩阵的起源**：矩阵最早来自于方程组的系数及常数所构成的方阵。这一概念由19世纪英国数学家凯利首先提出。
