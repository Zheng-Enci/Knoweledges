三种拟态玻璃导航按钮对比 🪟

没有背景图片 — 纯 CSS 构建高级玻璃质感

1 方法一 · box-shadow + RGBA 背景层

通过 inset/outset 阴影和半透明背景模拟玻璃厚度与悬浮感，性能最优，适合移动端和性能敏感场景

2 方法二 · 伪元素模糊层

通过 ::before 绘制渐变背景 + filter: blur() 自产自销模糊内容，不需要 backdrop-filter，兼容性更好

3 方法三 · 完整玻璃层叠方案（Apple Liquid Glass 风格 🍎）

backdrop-filter: blur(20px) saturate(180%) + 自建多层渐变背景 + 双向伪元素（表面高光 + 边缘折射色散），效果最逼真

💡 边缘折射提示： 方法三的 ::after 伪元素使用蓝→紫→粉渐变模拟玻璃切面的光色散。 Hover 按钮时折射效果会渐入显示。
