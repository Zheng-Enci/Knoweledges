## 1. 正交投影的定义 📐

> 本章建立正交投影的基本概念

### 1.1 什么是投影？

在线性代数中，**投影**是一种将向量空间中的向量映射到其子空间上的线性变换。想象一束光垂直照射到一个平面上，物体在平面上的影子就是该物体在这个平面上的**正交投影**。

设 $V$ 是一个向量空间，$W$ 是 $V$ 的一个子空间。一个**投影**是指一个线性映射 $P: V \to W$，满足以下**幂等性**（Idempotence）：

$$
P^2 = P \circ P = P
$$

这意味着对任意向量连续投影两次，结果与投影一次完全相同——因为投影后的向量已经在子空间 $W$ 中，再次投影不会产生变化。

### 1.2 什么是"正交"？

**正交投影**是投影的一种特殊形式，它要求投影方向与子空间 $W$ **垂直**（正交）。

设 $x$ 是向量空间中的任意向量，$x_W$ 是 $x$ 在子空间 $W$ 上的正交投影。那么：

$$
x - x_W \perp W
$$

即**原向量与投影向量的差**（称为**误差向量**或**残差向量**）与子空间 $W$ 中的所有向量都垂直。

**几何直观**：正交投影就是找到子空间 $W$ 中距离原向量 $x$ **最近**的那个点 $x_W$。这个最近点的特征正是：从 $x$ 到 $x_W$ 的连线垂直于 $W$。

## 2. 正交分解定理 🔀

> 本章阐述正交分解的存在性与唯一性

### 2.1 正交分解

设 $W$ 是 $\mathbb{R}^n$ 的一个子空间，$x$ 是 $\mathbb{R}^n$ 中的任意向量。则 $x$ 可以**唯一地**分解为：

$$
x = x_W + x_{W^\perp}
$$

其中：
- $x_W \in W$：$x$ 在子空间 $W$ 上的**正交投影**
- $x_{W^\perp} \in W^\perp$：$x$ 在 $W$ 的正交补空间 $W^\perp$ 上的分量
- $W^\perp$ 表示与 $W$ 中所有向量都正交的向量集合

### 2.2 唯一性证明思路

假设存在两种不同的分解：

$$
x = x_W + x_{W^\perp} = y_W + y_{W^\perp}
$$

移项得到：

$$
x_W - y_W = y_{W^\perp} - x_{W^\perp}
$$

等式左边属于 $W$，右边属于 $W^\perp$。一个向量既在 $W$ 中又在 $W^\perp$ 中，意味着它与自己正交，因此必须是零向量。故 $x_W = y_W$，$x_{W^\perp} = y_{W^\perp}$，分解唯一。

### 2.3 最近向量性质

正交投影 $x_W$ 是子空间 $W$ 中距离 $x$ **最近**的向量：

$$
\|x - x_W\| \leq \|x - w\|, \quad \forall w \in W
$$

这个距离 $\|x - x_W\| = \|x_{W^\perp}\|$ 就是向量 $x$ 到子空间 $W$ 的距离。

## 3. 投影到直线的公式 📏

> 本章推导一维子空间（直线）上的投影公式

### 3.1 向量投影到向量

设 $u$ 是一条直线 $L = \text{Span}\{u\}$ 的方向向量，$x$ 是待投影的向量。则 $x$ 在 $L$ 上的正交投影为：

$$
x_L = \frac{u \cdot x}{u \cdot u} \, u
$$

其中 $u \cdot x$ 表示向量 $u$ 与 $x$ 的**点积**（内积）。

### 3.2 公式推导

由于 $x_L$ 必须在直线 $L$ 上，可以表示为 $x_L = c \cdot u$（$c$ 为标量系数）。

根据正交性条件，$x - x_L$ 必须与 $u$ 垂直：

$$
(x - c \cdot u) \cdot u = 0
$$

展开点积：

$$
x \cdot u - c \cdot (u \cdot u) = 0
$$

解得：

$$
c = \frac{u \cdot x}{u \cdot u}
$$

因此：

$$
x_L = \frac{u \cdot x}{u \cdot u} \, u
$$

### 3.3 正交分量

原向量 $x$ 可以分解为：

$$
x = x_L + x_{L^\perp}
$$

其中正交分量为：

$$
x_{L^\perp} = x - x_L = x - \frac{u \cdot x}{u \cdot u} \, u
$$

**标量系数** $\frac{u \cdot x}{u \cdot u}$ 的物理意义：它表示 $x$ 在 $u$ 方向上的"阴影长度"与 $u$ 自身长度的比值，即 $x$ 在 $u$ 方向上占据了多少个 $u$ 的单位。

## 4. 投影到子空间的公式 📐

> 本章推导一般子空间上的投影矩阵表达

### 4.1 通过正规方程求解

设 $W$ 是 $\mathbb{R}^m$ 的一个子空间，$A$ 是一个 $m \times n$ 矩阵，其列向量构成 $W$ 的一组基，即 $W = \text{Col}(A)$。

对于任意向量 $x \in \mathbb{R}^m$，其正交投影 $x_W$ 可以通过求解以下**正规方程**（Normal Equation）得到：

$$
A^T A \, c = A^T x
$$

其中 $c$ 是系数向量。求出 $c$ 后：

$$
x_W = A \, c
$$

### 4.2 投影矩阵

当 $A$ 的列向量**线性无关**时（即构成 $W$ 的一组基），矩阵 $A^T A$ 可逆，此时可以直接写出**投影矩阵** $P$：

$$
P = A(A^T A)^{-1} A^T
$$

于是正交投影可以简洁地表示为矩阵乘法：

$$
x_W = P \, x = A(A^T A)^{-1} A^T x
$$

**重要说明**：投影矩阵 $P$ 只依赖于子空间 $W$，而不依赖于具体的基的选择。无论用哪组基构造矩阵 $A$，最终得到的投影矩阵 $P$ 都是相同的。

### 4.3 特殊情形验证

当 $W$ 是一条直线 $L = \text{Span}\{u\}$ 时，$A = u$ 是一个列向量。此时：

$$
A^T A = u^T u = u \cdot u, \quad A^T x = u^T x = u \cdot x
$$

代入投影矩阵公式：

$$
x_L = u \, (u^T u)^{-1} \, u^T x = \frac{u \cdot x}{u \cdot u} \, u
$$

这与第3节中向量投影到向量的公式完全一致，验证了一般公式的正确性。

## 5. 正交投影的核心性质 ⭐

> 本章总结正交投影作为线性变换的关键特性

设 $P$ 是到子空间 $W$ 的正交投影矩阵，则 $P$ 具有以下核心性质：

### 5.1 幂等性（Idempotence）

$$
P^2 = P
$$

投影两次等于投影一次。这是因为投影后的向量已经在 $W$ 中，再次投影不会产生变化。

### 5.2 自伴性（Self-Adjoint）

$$
P^T = P
$$

正交投影矩阵是对称矩阵。这一性质是正交投影与斜投影（Oblique Projection）的根本区别。

### 5.3 线性变换性质

正交投影 $T(x) = x_W$ 是一个**线性变换**，满足：

$$
T(x + y) = T(x) + T(y), \quad T(c \cdot x) = c \cdot T(x)
$$

### 5.4 值域与零空间

- **值域（Range）**：$\text{Range}(P) = W$，投影矩阵的列空间就是目标子空间
- **零空间（Null Space）**：$\text{Null}(P) = W^\perp$，被投影为零向量的恰好是正交补空间中的向量

### 5.5 特征值

投影矩阵 $P$ 的特征值只能是 $0$ 或 $1$：

- 特征值 $1$ 对应的特征空间：$W$（子空间本身）
- 特征值 $0$ 对应的特征空间：$W^\perp$（正交补空间）

这意味着 $P$ 相似于一个对角矩阵，其对角线上有 $\dim(W)$ 个 $1$ 和 $\dim(W^\perp)$ 个 $0$。

### 5.6 与单位矩阵的关系

$I - P$ 是到正交补空间 $W^\perp$ 上的正交投影矩阵：

$$
(I - P)^2 = I - 2P + P^2 = I - P
$$

且：

$$
x_{W^\perp} = (I - P) \, x = x - x_W
$$

## 6. 正交投影与最小二乘法 📊

> 本章揭示正交投影在数据拟合中的核心作用

### 6.1 线性方程组的近似解

在实际问题中，我们经常遇到**超定线性方程组**（方程个数多于未知数个数）：

$$
A \, x = b
$$

这类方程组通常**无精确解**。最小二乘法的核心思想是：寻找 $x$ 使得 $A \, x$ 尽可能接近 $b$，即最小化误差：

$$
\min_x \|A \, x - b\|^2
$$

### 6.2 几何解释

从几何角度看，$A \, x$ 的全体构成矩阵 $A$ 的列空间 $\text{Col}(A)$。最小二乘解就是找到列空间中距离 $b$ **最近**的向量。

而这个"最近向量"正是 $b$ 在 $\text{Col}(A)$ 上的**正交投影**：

$$
b_{\text{Col}(A)} = A(A^T A)^{-1} A^T b
$$

因此，最小二乘解满足：

$$
A \, x = b_{\text{Col}(A)}
$$

即残差向量 $b - A \, x$ 与 $\text{Col}(A)$ 正交，这正是正交投影的定义。

### 6.3 正规方程

将 $A \, x = A(A^T A)^{-1} A^T b$ 两边左乘 $A^T$，得到：

$$
A^T A \, x = A^T b
$$

这就是著名的**正规方程**（Normal Equation），它的解即为最小二乘解。

## 7. Python 代码示例 🐍

> 本章通过 Python 代码验证正交投影的计算

```python
import numpy as np                                       # 导入 NumPy 库，提供矩阵运算支持


"""计算向量在直线上的正交投影

参数:
    x: 待投影的向量，形状 (n,)
    u: 直线方向向量，形状 (n,)

返回:
    projection: x 在直线上的正交投影，形状 (n,)
    orthogonal_component: 正交分量 x - projection，形状 (n,)

示例:
    x = np.array([3, 4])
    u = np.array([1, 0])
    proj, orth = project_onto_line(x, u)
"""
def project_onto_line(x, u):
    # 计算点积 u·x 和 u·u
    u_dot_x = np.dot(u, x)                               # 计算 u 与 x 的点积，示例：u=[1,0], x=[3,4] → 3
    u_dot_u = np.dot(u, u)                               # 计算 u 与自身的点积，示例：[1,0]·[1,0]=1
    
    # 计算标量系数 c = (u·x) / (u·u)
    c = u_dot_x / u_dot_u                                # 计算投影系数，示例：3/1=3
    
    # 计算投影向量：c * u
    projection = c * u                                   # 投影向量 = 3 * [1,0] = [3,0]
    
    # 计算正交分量
    orthogonal_component = x - projection                # 正交分量 = [3,4] - [3,0] = [0,4]
    
    return projection, orthogonal_component              # 返回投影向量和正交分量


"""计算向量在子空间上的正交投影（通过投影矩阵）

参数:
    x: 待投影的向量，形状 (m,)
    A: 子空间的基矩阵，列向量为子空间的一组基，形状 (m, n)

返回:
    projection: x 在子空间上的正交投影，形状 (m,)
    P: 投影矩阵，形状 (m, m)

示例:
    x = np.array([1, 2, 3])
    A = np.array([[1, 0], [0, 1], [0, 0]])  # xy平面
    proj, P = project_onto_subspace(x, A)
"""
def project_onto_subspace(x, A):
    # 计算 A^T * A
    ATA = np.dot(A.T, A)                                 # 计算 Gram 矩阵，示例：A为xy平面基 → [[1,0],[0,1]]
    
    # 计算 (A^T * A)^(-1)
    ATA_inv = np.linalg.inv(ATA)                         # 求逆矩阵，示例：单位矩阵的逆仍是单位矩阵
    
    # 计算投影矩阵 P = A * (A^T * A)^(-1) * A^T
    P = np.dot(np.dot(A, ATA_inv), A.T)                  # 投影矩阵，示例：xy平面的投影矩阵
    
    # 计算投影向量
    projection = np.dot(P, x)                            # 投影 = P @ x，示例：[1,2,3] → [1,2,0]
    
    return projection, P                                 # 返回投影向量和投影矩阵


# ========== 测试示例 ==========

# 测试1：投影到直线
print("=" * 50)                                          # 打印分隔线
print("测试1：向量投影到直线")                            # 打印测试标题
print("=" * 50)                                          # 打印分隔线

x = np.array([3, 4])                                     # 定义待投影向量 x = [3, 4]
u = np.array([1, 0])                                     # 定义直线方向向量 u = [1, 0]（x轴）

proj, orth = project_onto_line(x, u)                     # 调用投影函数，数据流动：[3,4] → [3,0], [0,4]

print(f"原向量 x: {x}")                                  # 打印原向量
print(f"方向向量 u: {u}")                                # 打印方向向量
print(f"投影分量 x_L: {proj}")                           # 打印投影分量
print(f"正交分量 x_⊥: {orth}")                           # 打印正交分量
print(f"验证正交性 (x_⊥ · u): {np.dot(orth, u)}")        # 验证正交性，应为 0
print(f"验证分解 (x_L + x_⊥): {proj + orth}")            # 验证分解正确性
print()

# 测试2：投影到平面（xy平面）
print("=" * 50)                                          # 打印分隔线
print("测试2：向量投影到 xy 平面")                        # 打印测试标题
print("=" * 50)                                          # 打印分隔线

x = np.array([1, 2, 3])                                  # 定义待投影向量 x = [1, 2, 3]
A = np.array([[1, 0],                                    # 定义 xy 平面的基矩阵
              [0, 1],
              [0, 0]])

proj, P = project_onto_subspace(x, A)                    # 调用投影函数，数据流动：[1,2,3] → [1,2,0]

print(f"原向量 x: {x}")                                  # 打印原向量
print(f"投影矩阵 P:\n{P}")                               # 打印投影矩阵
print(f"投影分量 x_W: {proj}")                           # 打印投影分量
print(f"验证 P^2 = P:\n{np.dot(P, P)}")                  # 验证幂等性
print(f"验证 P^T = P:\n{np.allclose(P, P.T)}")           # 验证对称性
print(f"距离到平面: {np.linalg.norm(x - proj)}")         # 计算到平面的距离
```

## 8. 总结 📝

本节我们系统梳理了正交投影的核心概念：

| 概念 | 要点 |
|------|------|
| **定义** | 将向量映射到子空间且投影方向与子空间垂直的线性变换 |
| **正交分解** | $x = x_W + x_{W^\perp}$，分解唯一 |
| **直线投影** | $x_L = \frac{u \cdot x}{u \cdot u} \, u$ |
| **子空间投影矩阵** | $P = A(A^T A)^{-1} A^T$ |
| **核心性质** | 幂等性 $P^2=P$、自伴性 $P^T=P$、特征值为 0 或 1 |
| **最小二乘** | 正交投影给出了超定方程组的最佳近似解 |

🔴 **关键理解**：

- 正交投影的本质是"找最近点"——在子空间中找到距离原向量最近的点
- 幂等性 $P^2 = P$ 是投影矩阵的标志性特征
- 正交投影与最小二乘法的联系揭示了线性代数在数据分析中的核心作用
